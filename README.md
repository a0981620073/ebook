# 初級大人指南書｜互動電子書網站

這是單檔版 GitHub Pages 專案。

## 使用方式
1. 將 `index.html` 上傳到 GitHub Repository 根目錄。
2. Repository → Settings → Pages → Deploy from a branch → main / root → Save。
3. 等 1～3 分鐘後開啟 `https://你的帳號.github.io/ebook/`。

## 說明
- 內容已轉成 HTML 電子書，不是直接嵌入 PDF。
- 使用 CDN 載入 AOS 與 GSAP。
- 若要啟用聯絡表單寄信，請在 `index.html` 搜尋 `CONTACT_EMAIL` 並填入你的信箱。
